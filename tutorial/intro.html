Introducción para aprender HTML

Excelente, comencemos con algo muy básico. Estoy convencido de que todos aquí saben lo que es una página web; de hecho, es muy probable que estés realizando este curso en una página web, ¿no es así? Por supuesto que sí.

Para aquellos que no lo sepan, una página web y un sitio web son conceptos completamente distintos. A menudo, estos términos se utilizan indistintamente sin darles mucha importancia, pero es crucial distinguir entre ellos.

Definición de página web y sitio web

  • Página web.- Es un documento electrónico digital que puede contener elementos de texto, imágenes, vídeos, multimedia, enlaces, formularios, etc. Además una página web posee una URL o comúnmente llamado link. Por ejemplo: https://desarrolladoresweb.org/....
  • Sitio web.- Es un conjunto de páginas web, que están organizadas y estructuradas de tal manera que sea navegable, usable y estético para los usuarios y/o los motores de búsqueda. Además un sitio web tiene un nombre de dominio que es único en todo internet. Por ejemplo: www.desarrolladoresweb.org

Los navegadores web

En términos simples, los navegadores web son programas que facilitan la navegación por internet, como Firefox, Chrome, Safari, Opera, entre otros. Gracias a ellos, es posible acceder a cualquier página web introduciendo su dirección o URL, ya sea a través de un enlace o tecleándola directamente.

Introducción a HTML Y los navegadores web

Es correcto decir que las páginas web están construidas con código HTML; dicho código es procesado por el navegador para mostrar un contenido que cualquier usuario pueda entender.

Visualizar el código HTML de una página web es tan sencillo como hacer clic derecho con el ratón en cualquier parte de la página y seleccionar «Ver código fuente de la página». También es posible mediante la opción «Inspeccionar». Así, podemos descubrir que detrás de cada elemento hay un código HTML.

Interacción de HTML y el navegador web

Te explicaré de manera muy sencilla cómo las páginas web están disponibles en todo momento y lugar, ya que asumo que estás iniciándote en este tema.

Los códigos HTML se pueden escribir en cualquier editor de texto, aunque es preferible utilizar un editor de código (Abordaremos este tema en otra lección). Una vez escritos, los códigos se guardan y para que estén disponibles en internet, se almacenan en un servidor que permanece conectado a la red constantemente.

Cualquier usuario puede solicitar a un servidor, a través de un navegador web, que muestre el contenido de un documento HTML. El servidor accede rápidamente al documento, interpreta el código y muestra una página con contenido comprensible, incluyendo textos, imágenes, listas, enlaces y otros elementos definidos previamente en el documento.

Cómo es un código HTML

Un código HTML se basa en etiquetas, conocidas como ‘tags’ en inglés, y es el tema que desarrollaremos a lo largo del curso. Aparentemente, está lleno de signos de menor y mayor, entre los cuales se colocan los nombres de las etiquetas para definir el tipo de elemento que se mostrará en la página web a través del navegador.
